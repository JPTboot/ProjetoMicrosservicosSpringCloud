############ DIRETÓRIOS PRINCIPAIS ####

/bin/ -> contém executáveis do linux
/boot/ -> ficheiros para iniciar o sistema
/dev/ -> contém arquivos de biblioteca, dispositivos, drivers
/etc/ -> personalização do sistema
/home/ -> diretório comum. É a pasta pessoal.
/lib/ -> bibliotecas essencias do sistema
/media/ -> diretório de montagem de dispositivos
/mnt/ -> diretório de montagem de dispositivos
/opt/ -> programas não oficias do sistema mas que eu quero instalar
/sbin/ -> armazena executáveis
/srv/ -> serviços oferecidos pelo sistema
/tmp/ -> para arquivos temporários, de transferência
/usr/ -> a segunda hierarqui de sistema, onde ficam os usuários do sistema
/var/ -> onde ficam os históricos, de emails, cache, impressoras
/root/ -> usuário root, contém o usuário com mais privilégios do sistema
/proc/ -> diretório virtual controlado pelo kernel. Tem informações sensíveis do kernel
